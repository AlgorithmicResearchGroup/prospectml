Deploy Your AI Agent Army in Minutes

Get started with just a few commands in your Kubernetes cluster

Terminal - ProspectML Installation
# Add ProspectML helm repository
helm repo add prospectml https://charts.prospectml.com

# Install ProspectML with your Anthropic API key
helm install prospectml prospectml/prospectml \
  --set aiProvider.anthropic.apiKey=YOUR_API_KEY

# Submit your first ML task
prospectml run "Predict equipment failures 72 hours in advance using \
  vibration sensors, maintenance logs, and weather data"

# Output: Deploying 47 specialized agents...
# Output: Model deployment complete - optimized for your requirements
